## ✨ 主要功能

### 1️⃣ 文件夹选择

- **输入文件夹**: 包含 `wos.txt` 和 `scopus.csv` 的文件夹
- **输出文件夹**: 可选，留空则输出到输入文件夹

### 2️⃣ 参数配置

- **年份范围**: 设置起始年和结束年
- **目标语言**: 选择要保留的语言（English, Chinese等）
- **清洗规则**: 选择机构清洗的强度

### 3️⃣ 功能开关

- **AI智能补全**: 补全机构地理信息
- **机构名称清洗**: 合并重复机构
- **生成分析图**: 生成文档类型分布图

### 4️⃣ 实时反馈

- **处理日志**: 实时显示处理进度
- **状态栏**: 显示当前状态
- **进度提示**: 按钮状态变化

## 📋 使用流程

```
1. 双击 启动GUI.command
        ↓
2. 选择输入文件夹（包含 wos.txt 和 scopus.csv）

7. 查看实时日志
        ↓

```

## 💡 使用技巧

### 技巧1: 快速重复处理

如果需要对同一数据用不同参数处理多次：
1. 第一次处理完成后不要关闭GUI
2. 修改参数
3. 再次点击 "开始处理"

### 技巧2: 查看详细日志

日志窗口支持：
- 滚动查看
- 复制文本
- 清空重置

### 技巧3: 输出到不同文件夹

如果想保留多个版本的处理结果：
1. 第一次: 输出文件夹留空
2. 第二次: 指定新的输出文件夹
3. 每次处理结果互不影响

## 🆘 常见问题

### Q1: 双击 `启动GUI.command` 后没反应？

**A**: 首次运行需要安装依赖，请稍等30秒-1分钟。如果仍无反应：
```bash
cd /Users/menglinghan/Desktop/MultiDatabase
chmod +x 启动GUI.command
./启动GUI.command
```

### Q2: 提示缺少 customtkinter？

**A**: 手动安装：
```bash
pip3 install customtkinter
# 或
pip3 install --break-system-packages customtkinter
```

### Q3: GUI界面显示异常？

**A**: 确保使用 macOS 10.14 或更高版本，并更新 Python 到 3.6+

### Q4: 处理失败？

**A**: 检查日志窗口中的错误信息，常见原因：
- 输入文件夹缺少 wos.txt 或 scopus.csv
- 文件编码不是 UTF-8
- 磁盘空间不足
