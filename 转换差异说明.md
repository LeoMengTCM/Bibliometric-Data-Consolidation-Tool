# Scopus转WOS格式转换差异说明

## ✅ 已完美匹配的格式

### 1. 文件结构
- ✅ FN、VR、PT连续无空行
- ✅ 记录之间有空行分隔
- ✅ 最后ER和EF之间有空行

### 2. 字段格式
- ✅ **TI（标题）**：在80字符处自动换行
- ✅ **DE（作者关键词）**：在80字符处换行，`vitamin B-12`格式正确
- ✅ **AB（摘要）**：`F:M ratio`（无空格），`+/-`（不是±符号）
- ✅ **AU（作者）**：`Lenti, MV`格式（无空格和点号）
- ✅ **C1（机构地址）**：`[姓, 名]`格式，一级机构在前
- ✅ **C3（一级机构）**：智能提取并去重

## ⚠️ 固有差异（数据源限制）

### 1. ID字段（索引关键词）

**WOS原生：**
```
ID ATROPHIC GASTRITIS; DIAGNOSIS; CLASSIFICATION
```
- 3个核心词
- 由WOS自动生成的"Keywords Plus"

**转换后：**
```
ID chromogranin A; gastrin; Ki 67 antigen; parietal cell antibody; adult; amnesia; ...（超长列表）
```
- 完整的Scopus Index Keywords
- Scopus数据库内部生成的索引词

**原因：**
- WOS的"Keywords Plus"是基于引用文献标题词自动提取的，无法从Scopus重新生成
- Scopus的Index Keywords更详细，对检索和分析有价值
- 建议保留，或可根据需要手动编辑为简短关键词

### 2. AF字段（作者全名）

**WOS原生：**
```
AF Anderloni, Andrea
```

**转换后：**
```
AF Anderloni, Andrea A
```

**原因：**
- Scopus和WOS对同一作者的记录略有不同
- Scopus提供的中间名缩写可能更完整
- 建议保留Scopus的信息，更有利于作者识别

### 3. C3字段（一级机构）

**WOS原生：**
```
C3 IRCCS Fondazione San Matteo; University of Pavia; IRCCS Fondazione San
   Matteo; University of Pavia; IRCCS Fondazione San Matteo; University of
   Pavia; IRCCS Fondazione San Matteo
```
- 机构名重复出现（每个作者对应一次）

**转换后：**
```
C3 University of Pavia; IRCCS Fondazione Policlinico San Matteo
```
- 去重后的机构列表

**原因：**
- WOS重复计数是为了统计每个作者的机构归属
- 转换后去重更适合机构合作分析，避免同一文章重复计数同一机构
- **优势：** 转换后的格式在机构分析中更合理

### 4. 缺失字段

以下字段Scopus CSV不提供，无法转换：

- **EM**：作者邮箱地址
- **RI**：ResearcherID（Clarivate的研究者标识符）
- **OI**：ORCID（开放研究者与贡献者身份识别码）

**影响：** 极小，这些字段在文献计量分析中很少使用

### 5. CR字段（参考文献）

**WOS原生：**
```
CR Green R, 2017, NAT REV DIS PRIMERS, V3, DOI 10.1038/nrdp.2017.40
```
- 含作者首字母、完整DOI、精确卷号

**转换后：**
```
CR Green, 2017, NAT REV DIS PRIMERS, V3
```
- 简化格式，缺少作者首字母和DOI

**原因：**
- Scopus References字段格式与WOS不同
- 作者首字母可能缺失
- 卷号可能不精确
- DOI信息未包含在Scopus References字段中

**影响：** 中等
- ✅ 适用：共被引分析、引文网络分析（基于年份和期刊）
- ⚠️ 注意：精确引文链接可能受影响
